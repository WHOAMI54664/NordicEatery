# Mike’s Food Starter

Starter website for ordering Polish street food: Maczanka Krakowska and Knysza.

## Run locally

```bash
npm install
npm run dev
```

Open:

```txt
http://localhost:3000
```

## Current features

- Home page
- Menu page
- Cart with localStorage
- Checkout demo form
- Kitchen tablet order board at `/kitchen`
- Responsive UI
- Warm Polish food branding
